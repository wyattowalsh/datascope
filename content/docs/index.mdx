# Welcome to DataScope

**DataScope v2.7** is a professional-grade data exploration, visualization, and analytics platform designed for structured data formats. Built with modern web technologies, it provides enterprise-level features for parsing, analyzing, and transforming data.

- **Auto-Save**: Persistent state every 2 seconds to IndexedDB

- **Crash Diagnostics**: Structured telemetry for debugging

### High-Performance Architecture
- **Web Workers**: Off-thread parsing prevents UI blocking
- **Virtualized Rendering**: Handle 10,000+ tree nodes at 60 FPS
- **Canvas Graphs**: Optimized 2D graphs for 5,000+ nodes
- **Level-of-Detail**: 3D graphs maintain performance with complex scenes

- **Full JSONPath Support**: Execute complex queries on any data
- **Smart Suggestions**: Context-aware query recommendations
- **Saved Queries**: Persist favorites via URL or storage
- **Live Preview**: Real-time results with path highlighting
